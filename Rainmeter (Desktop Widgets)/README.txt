1. Instal Rainmeter: https://www.rainmeter.net/
2. Install all of the skin in the "Install" folder
3. Move folders in "Skins" folder to your rainmeter "Skin"s folder (C:\Users\'name'\Documents\Rainmeter\Skins)
4. *Optional* Open the "Rainmeter.ini" file in the "Rainmeter (Desktop Widgets)" folder, copy contents, open the "Manage Rainmeter" menu, navigate to the "Layouts" tab, click on any saved layout then click "Edit", replace current contents with the contents you copied, save, close the file, click "Load"
